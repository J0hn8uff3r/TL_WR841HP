IPsec Policy Settings Help

You could simply setup IPsec Policy referring to the following instructions.

  • Policy Name - Enter the name for the IPsec Policy.
  • Local Subnet - Enter the local (LAN) subnet and mask.(ex. 192.168.0.0/24)
  • Peer Subnet - Enter the Peer subnet and mask.
  • Peer Gateway - Enter the Peer gateway or domain.
  • Negotiation Mode - Here you could find two options, IKE Negotiation Mode and Manually Mode.You are recommended to select the default mode-- IKE Negotiation Mode, and all the parameters could be negotiated automatically according to IKE.If Manually Mode is selected, you should manually setup the Encryption and SPI parameters according to the instructions.
  • Security Protocol - Select the Security Protocol from the drop-down menus. Here you could find AH (Authentication Header) and ESP (Encapsulation Security Payload) protocols.
  • Authentication Algorithm - Select the Authentication Algorithm for IPsec connection. You could also keep the default value "Auto".
  • Encryption Algorithm - Select the Encryption Algorithm for IPsec connection. You could also keep the default value "Auto".
  • In SPI - It is for direction in SPI parameter setting in manual mode, and the parameter must be same as peer Out SPI.
  • In Authentication Key - It is for direction in authentication key setting in manual mode, and the key must be same as peer Out Authentication Key.
  • Out SPI - It is for direction out SPI parameter setting in manual mode, and the parameter must be same as peer In SPI.
  • Out Authentication Key - It is for direction out authentication key setting in manual mode, and the key must be same as peer In Authentication Key.
  • PFS Group - Select the PFS property from the drop-down menus that should be the same for the local and remote endpoints. If you select "None" for local endpoint, any value is accepted for remote endpoint.
  • Lifetime - Manually enter the number of seconds for the IPsec Lifetime. The default value is 28800.
  • Enable - Enable or Disable current policy.
